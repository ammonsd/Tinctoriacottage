Photo Content input contains the following details:

 

  • Picture File Name.   Must have a “GIF” or “JPG” extension and stored in www.tinctoriacottage.com/photos.  If the file does not exist in the photo folder, an error message will be displayed and the build process terminated.
  • Photo Caption.  This optional text will appear beneath the photo in bold font and can contain HTML formatting tags.
  • Photo Description.  This optional text will appear under the photo caption, or beneath the photo if a caption is not provided, in a normal font and can contain HTML formatting tags.

 

 

Links Content input contains the following details:

 

  • Link Group Name.  A short description of the link group.  Each group only needs to be entered once.  When left blank, the previously entered group description will be assumed. 
  • Link Address.  The web site address with or without the http: prefix.  The link will appear under the Link Group Name on the web site.
  • Link Description.  A short description of the link that will display on the web site.  If a description is not provided the link address will be used.
